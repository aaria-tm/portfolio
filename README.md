# portfolio- aarya londhe

A sleek, responsive personal portfolio website.

## ✨ Features
*   **About Me:** A brief introduction to my background and goals.
*   **Services / What I Do:** Details on my core skills and professional offerings.
*   **Resume:** Added resume for more information about me.
*   **Projects:** A showcase of my latest work with links.
*   **Tech Stack:** A list of technologies I use (HTML, CSS, JavaScript, etc.).
*   **Contact:** Direct links to reach me via email or social platforms.

## 🛠️ Built With
*   **HTML5:** Structural foundation.
*   **CSS3:** Custom styling with a curated brown palette and responsive layouts.
*   **JavaScript:** Interactive elements and smooth scrolling.
